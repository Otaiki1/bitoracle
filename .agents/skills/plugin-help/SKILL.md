# Stacks Plugin Help

Welcome to the Stacks Claude Plugin! This tool is designed to help you build, test, and deploy smart contracts on the Stacks blockchain using Clarity.

## Available Commands

*   `/stacks:init`: Initialize a new Stacks project or sync an existing one.
*   `/stacks:help`: Display this help message.
*   `/stacks:expert-advice`: Get technical guidance on Clarity and Stacks architecture.
*   `/stacks:update-docs`: Refresh the documentation index for latest information.

## Core Capabilities

1.  **Contract Deployment**: Use `/stacks:deploy-contract` for safe, guided deployments.
2.  **Testing**: Use `/stacks:run-tests` for TDD and error diagnosis.
3.  **Dev Environment**: Use `/stacks:start-dev-server` to start Clarinet devnet and frontend.

## Getting More Help

If you're stuck, try asking Claude for "Expert Advice" or consult the [Stacks Documentation](https://docs.stacks.co).
