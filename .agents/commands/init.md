# /stacks:init

Initialize or synchronize a Stacks project.

## Usage
- Run this command when starting a new project.
- Use it to ensure all Stacks-related configurations are present.

## Effects
- Configures `Clarinet.toml`.
- Sets up standard directory structure (`contracts/`, `tests/`, `deployments/`).
- Installs necessary dependencies (Stacks.js, Clarinet SDK).
